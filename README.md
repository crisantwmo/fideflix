# Fideflix - Sistema de Gestión de Streaming

Fideflix es una aplicación de escritorio desarrollada en **Java** como proyecto universitario. El sistema permite la gestión de usuarios y contenido multimedia, simulando las funciones básicas de una plataforma de streaming.

## Características
* **Gestión de Usuarios:** Registro e inicio de sesión con almacenamiento persistente.
* **Interfaz Gráfica:** Construida con Java Swing para una experiencia de usuario intuitiva.
* **Persistencia de Datos:** Uso de archivos `.dat` para mantener la información de los usuarios.

## Tecnologías utilizadas
* **Lenguaje:** Java 17+
* **Gestor de dependencias:** Maven
* **GUI:** Java Swing / AWT

## Instalación y Ejecución
1. Clona el repositorio: `git clone https://github.com/crisantwmo/fideflix.git`
2. Abre el proyecto en **Apache NetBeans**.
3. Asegúrate de que las dependencias de Maven se carguen correctamente.
4. Ejecuta la clase `FideBank.java` (o la clase principal correspondiente).
